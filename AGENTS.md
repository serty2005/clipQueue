# AGENTS.md

## Основные правила работы

1. Все ответы, комментарии в коде, логи и сообщения должны быть строго на русском языке.
2. Нельзя придумывать детали реализации, связанные с внешними системами или неоднозначными требованиями. Если информации недостаточно, нужно задать уточняющие вопросы и дождаться ответа, прежде чем писать код.
3. Комментарии в коде должны быть только актуальными. Не использовать пометки вроде "ОБНОВЛЕНО" или "ДОБАВЛЕНО". Комментировать только реальную текущую функциональность.
4. Читай файлы кода и генерируй код только с использованием python. В среде установлен pip, поэтому можешь установить необходимые для чтения и редактирования кода библиотеки. Для корректного чтения используй UTF-8 кодировку в консоли.
5. Актуальная документация ко всем библиотекам доступна через Context7-MCP в DOCKER MCP и в интернете
6. Если используешь Powershell, то всегда принудительно указывай кодировку UTF-8 для чтения-записи, потому что мы работает на русскоязычной ОС и кириллица будет отображаться в mojibrake.
7. Используй Docker-MCP Playwright для автоматического тестирования UI-изменений.
